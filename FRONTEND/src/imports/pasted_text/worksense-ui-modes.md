Design a modern SaaS productivity web application UI for WorkSense – Smart Productivity Intelligence System that supports two user modes: Student Mode and Corporate Mode. Both modes use the same underlying data and functionality, but the UI, labeling, tone, and experience must feel different to match the user’s context.
